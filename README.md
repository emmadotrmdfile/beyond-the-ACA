## README

This repository contains original datasets and accompanying analysis related to gender composition in U.S. state legislatures and gender-specific insurance policies from 2010â€“2024. The datasets include:

- **State-level legislative composition**: the proportion of women legislators in each state.  
- **Policy datasets**: gender-specific insurance mandates for women, men, and transgender-friendly policies, as well as negative/restrictive policies.  
- **Cosponsor information**: detailed data on cosponsorship demographics for these policies.

The repository also includes:

- **Working paper** outlining research questions and results  
- **Figures and regression outputs** used in the analysis  
- **Codebook** explaining all variables  
- **Rmd files** used for data cleaning, analysis, and visualization
